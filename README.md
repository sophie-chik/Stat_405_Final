Instructions on installing LFS: 
If you have brew: 
        Open terminal and run: 
        $ brew install git-lfs
If not, go on: 
        https://git-lfs.com/ 
        Download and follow instructions 

Again on your terminal, run:
        $ git lfs install 
Should say "> Git LFS initialized." 
You should be good to go!
